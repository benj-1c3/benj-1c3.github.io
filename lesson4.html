How to Code a Simple Calculator in Java

In this tutorial, you will learn how to create a simple calculator using Java with a Scanner for user input and an operator system for basic calculations.

Step 1: Import Required Packages

First, import the Scanner class for taking user input.

import java.util.Scanner;

Step 2: Create the Main Class

Define the main class and method where the program execution will begin.

public class Calculator {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        System.out.print("Enter first number: ");
        double num1 = scanner.nextDouble();
        
        System.out.print("Enter an operator (+, -, *, /): ");
        char operator = scanner.next().charAt(0);
        
        System.out.print("Enter second number: ");
        double num2 = scanner.nextDouble();
        
        double result;
        
        switch (operator) {
            case '+':
                result = num1 + num2;
                break;
            case '-':
                result = num1 - num2;
                break;
            case '*':
                result = num1 * num2;
                break;
            case '/':
                if (num2 != 0) {
                    result = num1 / num2;
                } else {
                    System.out.println("Error: Division by zero is not allowed.");
                    return;
                }
                break;
            default:
                System.out.println("Error: Invalid operator.");
                return;
        }
        
        System.out.println("Result: " + result);
        scanner.close();
    }
}

Step 3: Explanation

  • We use the Scanner class to take user input.
  • The user enters two numbers and an operator.
  • A switch statement processes the operator and performs the calculation.
  • The result is displayed to the user.
  • Division by zero is checked to prevent errors.

Step 4: Run the Program

Compile and run the program in your Java environment. Try different inputs to test its functionality.
