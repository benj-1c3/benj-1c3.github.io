Java Basics Quiz

Test Your Java Knowledge

1. What is the correct syntax to print "Hello, World!" in Java?
print("Hello, World!"); (Incorrect: Java uses System.out.println() for output.)
System.out.println("Hello, World!");
echo("Hello, World!"); (Incorrect: echo() is used in PHP, not Java.)
2. Which data type is used to create a variable that should store text?
String
int (Incorrect: int is used for integer values, not text.)
boolean (Incorrect: boolean stores true or false values, not text.)
3. How do you create a variable with the floating number 2.8?
float num = 2.8f;
float num = 2.8; (Incorrect: A float must have an 'f' suffix in Java.)
int num = 2.8; (Incorrect: int only stores whole numbers.)
4. Which keyword is used to define a class in Java?
class
Class (Incorrect: Java is case-sensitive; 'class' must be lowercase.)
define (Incorrect: 'define' is used in other languages, not Java.)
5. What is the default value of a boolean variable?
false
true (Incorrect: Java assigns 'false' as the default value for booleans.)
null (Incorrect: Only objects can have null as a default value.)
